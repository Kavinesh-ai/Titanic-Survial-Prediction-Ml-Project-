# Titanic Survival Prediction

## Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning and Logistic Regression.

## Technologies Used
- Python
- Pandas
- Scikit-Learn

## Features Used
- Passenger Class (Pclass)
- Age

## Workflow
1. Load Dataset
2. Data Preprocessing
3. Train-Test Split
4. Logistic Regression Model
5. Prediction and Evaluation

## Results
The model predicts passenger survival and is evaluated using Accuracy Score.
